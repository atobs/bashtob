# Installation

    # we need -g flag because it has binaries
    sudo npm install -g bashtob

## bashtob

view posts on boards:

    bashtob /b
    bashtob 33000
    bashtob 33000!

## writetob

write posts or replies on boards

    writetob /b
    writetob 33000

## tailtob

subscribe to see new replies and updates happening on atob

    tailtob
